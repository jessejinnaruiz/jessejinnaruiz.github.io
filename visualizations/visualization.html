Project Overview

This analysis examines displacement patterns in Austin, Texas through 2020, visualizing how urban development and gentrification have affected different neighborhoods. The interactive dashboard allows users to explore demographic changes, housing costs, and displacement indicators across the city.

Key Insights

Geographic Patterns

The visualization reveals clear geographic clustering of displacement, with certain neighborhoods showing significantly higher risk factors than others.

Demographic Impact

Analysis shows disproportionate effects on specific demographic groups, highlighting the intersection of housing and social equity issues.

Policy Implications

The data provides actionable insights for urban planning and policy decisions to address displacement concerns proactively.

Data Sources & Methodology

This analysis combines multiple data sources to create a comprehensive view of displacement patterns:

  • Census Data: American Community Survey demographic and housing data
  • Housing Records: Property assessments and sales data from Travis County
  • Economic Indicators: Income, employment, and cost of living metrics
  • Geographic Data: Austin city boundaries and neighborhood classifications

Analysis Framework

The displacement analysis uses a multi-factor approach examining:

  • Housing cost burden and affordability ratios
  • Demographic composition changes over time
  • Economic mobility and income displacement
  • Geographic accessibility to resources and services

Technical Implementation

This visualization was built using:

  • Tableau Public: Interactive dashboard development and hosting
  • Data Processing: Python pandas for data cleaning and preparation
  • Geospatial Analysis: PostGIS for geographic calculations and spatial joins
  • Data Integration: Custom ETL pipeline to combine multiple data sources

The interactive features allow users to filter by time period, demographic groups, and geographic areas to explore specific aspects of the displacement patterns in detail.
